Get Start

  1. 安装 Angular Material 和 Angular CDK

    npm install --save @angular/material @angular/cdk

  2. Animations

    一些 Material 维护依赖 Angular animations 模块，如果需要动画效果，那么需要安装它。 
    npm install --save @angular/animations
    注意 animation 并不被所有的浏览器支持，想要支持老旧的浏览器，还需要引入 polyfill. 
    import {{ '{' }} BrowserAnimationsModule {{ '}' }} from '@angular/platform-browser/animations'
@NgModule({{ '{' }}
      ... imports:[BrowserAnimationsModule], ...
{{ '}' }})
    如果你想使用自己的动画库，则将上面的模块替换为 NoopAnimationsModule

  3. 使用组件

    需要使用哪个组件，导入它即可。 注意：组件的导入必须在 `BrowserModule` 之后
    import {{ '{' }}MatButtonModule{{ '}' }} from '@angular/material'
@NgModule({{ '{' }}
      ... imports:[MatButtonModule], ...
{{ '}' }})

  4. 主题

    必须为应用使用一个主题；如果是通过 Angular CL 创建的项目，则在 `styles.css` 文件中添加以下代码来添加官方的主题 
    @import "~@angular/material/prebuilt-themes/indigo-pink.css";
    否则，直接在 index.html 中使用 <link> 标签。
    如果想使用 自定义的主题，可以参考官方给出的示例
    1. src/theme/theme.scss + .angular-cli.json 主题的设置和入口配置
    2. app/core/header 中的开头, app.component 中 class 的切换
    3. app.component.css 中正文内容的设置, OverlayContainer.getContainerElement().classList.add('my-dark-theme');

  5. 手势(移动端)支持

    为了获得一些组件手势的支持，还需要安装 HammerJS
    npm install --save hammerjs
    安装完成之后，在应用的入口文件 `main.ts` 或项目的核心模块中 `core.module.ts` 中添加 `import 'hammer.js'`

  6. (可选) Material Icons

    如果想使用官方的 `mat-icon` 组件，那么需要引用官方的字体库，在 index.html 中 导入 <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
    由于网络原因，你可以将其下载下来，然后在 styles.css 中引用它们，具体查看 src/styles.css 文件

  7. (附录) SystemJS 配置

    如果你的项目是使用 SystemJS 进行模块化管理， 那么你还需要将 @angular/material 和 @angular/cdk 添加到 SystemJS 的配置文件中

指导

  1. 主题

    主题是 Angular Material 组件的 颜色 的集合。 通常一个主题包含了一级调色板，它们代表了不同的含义
    • primary: 屏幕及大部分的组件
    • accent: 交互的按钮，响应的组件
    • warn: 错误的状态
    • foreground: 文字 或 图标的颜色
    • background: 组件的背景颜色
    Angular Material 内置了以下的主题, `deeppurple-amber.css`,`indigo-pink.css`, `pink-bluegrey.css`, `purple-green.css` ，你可以在 `styles.css` 文件中通过以下方式使用它 
    @import "~@angular/material/prebuilt-themes/indigo-pink.css";
    注意: 如果应用的内容没有放在 `mat-sidenav-container` 组件中，那么就需要为包裹内容的控件(比如 body ) 添加一个 `mat-app-background` 的样式类，以便正常能正确地使用主题背景
    其它 自定义主题、多主题、延迟加载、 更换 Component 主题 等参阅文档。

  2. 排版

    排版遵循 Material.io 文档。 包含了以下多种级别，每个级别都定义了 `font-size`,`line-height`,`font-weight`
    • `display-4`,`display-3`,`display-2`,`display-1` 用在页面的顶部，比如 <header>
    • `headline`=<h1>, `title`=<h2>, `subheading-2`=<h3>, `subheading-1`=<h4>
    • `body-1` 基本的正文字体，`body-2` 更粗的正文字体
    • `caption` 更小的正文字体，或者 错误提示信息
    • `button` 按钮
    • `input` 输入框
