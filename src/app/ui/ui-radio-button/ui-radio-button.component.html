Radio button

提供了类似原生 <input type="radio"> 的功能
选项 1 选项 2
所有的 radio-buttons 都使用了相同的 name 属性, 一次只能选择一个
Radio-button Label
label 使用的是 <mat-radio-button> 组件，你可以通过 `labelPosition`="before|after" 来控制标签显示的位置。 如果不希望 label 靠着按钮，那可以使用 `arial-label` 指定一个特殊的 label.

Radio groups

当使用按钮组 <mat-radio-group> 时，所有的按钮都必须写在它里面。

1：mat-radio-group 具有一个 value 属性，表示当前所选的值。

2：内部的按钮将会继承按钮组的名字 name

<mat-radio-group> 支持 @angular/form 中的 FormsModule 和 ReactiveFormModule.
