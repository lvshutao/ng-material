Input

通常与 <mat-form-field>, <mat-error>, <mat-hint> 搭配使用。

支持的类型有: date, datetime-local, email, month, number, password, search, tel, text, time, url, week

最基本的用法

所有能够在 <input> 和 <textarea> 的 attribute，也都可以在 <mat-form-field> 内部的组件（包括 Angular 指令如 ngModel, formControl ）中使用，
下面的 types 都能够被用于 matInput 中: date, datetime-local, email, month, number, password, search, tel, text, time, url, week
