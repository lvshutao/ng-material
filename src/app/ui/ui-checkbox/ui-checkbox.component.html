Checkbox

<mat-checkbox> 与原生的 <input type="checkbox"> 一样的功能。 使用非常简单，比如 <mat-checkbox>Check me!</mat-checkbox>
点我!
Checkbox label
mat-checkbox 的内容即是标签，你可以通过为 mat-checkbox 添加 `labelPosition`="after|before" 来控制它的位置。，如果你想不要标签，可以设置 `aria-lable` 属性。
mat-checkbox 与 @angular/forms 组合，它支持 FormsModule 和 ReactiveFormsModule.
Indeterminate state
与原生一样，你可以通过添加 [checked]=boolean 来设置它是否处于选中的状态。
你可以添加 (click) = "$event.stopPropagation()" 来阻止它继续向外冒泡。
