GridList

用于相似的数据展示，尤其是图片、类似相册等；
指令为 mat-grid-list, 其它属性
  • cols(必须)：用来表示多少列，之后会根据数据量自动算出需要多少行

  • 行高 rowHeight

    • 固定值：默认单位为 px, 支持 em, rem
    • ratio：单元格的宽高比，只支持比例的样式(如 4:3)，不支持小数点
    • fit: 根据行数自动计算，此时你必须指定 grid-list 的高度
    如果不设置，那么默认采用 1:1 的比例显示
  • gutterSize 默认为 1px

其它指令

  • mat-grid-tile 单元格，具有 colspan, rowspan 属性，用来设置跨行跨列
  • mat-grid-tile-header
  • mat-grid-tile-footer
mat-grid-tile-header 1 2 3 4 mat-grid-tile-footer
