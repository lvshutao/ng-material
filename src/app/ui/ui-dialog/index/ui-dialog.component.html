dialog

`MatDialog` 服务用于打开对话框.
`open` 方法来打开机一个对话框，并返回一个 `MatDialogRef` 实例。你可以通过这个实例来关闭对话框，或者在对话框关闭时接收到一个通知。
注意：你必须将模态框注册到模型类中，如 @NgModule.entryComponents:[UiDialogConfirmComponent] 中，否则会一直报错
// 打开一个对话框
let dialogRef = dialog.open(UserProfileComponent,{{'{'}}
    height: '400px',
    width: '600px'
{{'}'}})
// 订阅关闭通知
dialogRef.afterClosed().subscribe(result => {{'{'}}
  console.log('Dialog result:',result);
{{'}'}})
// 关闭
dialogRef.close('Pizzal');

// 传递数据
const dialogRef = dialog.open(YourDialog, {{'{'}}data:'your data'{{'}'}});
// 接收数据
constructor(@Inject(MD_DIALOG_DATA) public data:any)
1. 将 MatDialogRef 注入到你的模态框中; 2. 使用 @Inject(MAT_DIALOG_DATA) 来接收传递到模态框的数据； 比如 src/app/ui/ui-dialog/confirm/ui-dialog-confirm.component.ts
模态框中使用 mat-dialog-title, mat-dialog-content, mat-dialog-actions, mat-dialog-close 来构建模态内容
当模态框打开时，它会自动聚集到第一个控件上，但你可以通过设置 tabindex="-1" 来取消它

常见错误

No component factory found for ProjectNewComponent. Did you add it to @NgModule.entryComponents?
没有将组件设置到模块的 entryComponents 中
