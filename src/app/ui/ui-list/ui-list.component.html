List

用途：一般用于列表，包含多行列表、带图标列表、带头像列表、密集列表和多段列表。

外部指令有 <mat-list> 和 <mat-nav-list>， 内部的指令为 <mat-list-item>

<mat-list> 只提供了 Material 的样式，本身并不提供行为
Peper Salt Paprika
  1. Navigation lists
    <mat-nav-list> 主要用于导航列表，简单的项可以使用 `mat-list-item` 属性。如 <a mat-list-item href="..." *ngFor="let link of links">...</a>
    Item1
    对于复杂的子项，需要将其它的内容包裹在 <mat-list-item> 内部，比如添加 mat-list-item > a + button > mat-icon
    Item2
  2. Selection lists

    <mat-selection-list> 选择列表主要用于交互，它使用到的子项是 <mat-list-option>

    {{ shoes }}

    选择: {{ shoes.selectedOptions.selected.length }}

    注意：此时的 mat-list-option 不能够包含任何交互的控件，比如 button, a

  3. 多行列表展示
    注意：列表显示的行都需要添加 matLine 属性.比如一个 p[matLine] 就是一个项。

    两行展示

    mat-list-item > h3[matLine] + p[matLine]

    {{message.from}}

    {{message.subject}} -- {{message.content}}

    三行展示

    mat-list-item > h3[matLine] + p[matLine] + p[matLine]

    {{message.from}}

    {{message.subject}}

    {{message.content}}
